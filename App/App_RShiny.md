# R Shiny Application

This folder contains the R Shiny application used to deploy the equity report generator as an interactive web application.

The equity reports can be generated locally using the R Markdown workflow. To make the process more accessible, the report generation pipeline was integrated into a Shiny application and deployed online.

The application allows users to:

- Enter a stock ticker
- Generate an automated equity research report
- Access the generated PDF report
