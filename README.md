# Automated Equity Research Report Generator

## Overview

End-to-end equity research reporting system that combines financial data extraction, analysis, and automated report generation.

The project retrieves financial data from external APIs using R, processes and analyses market information, and generates structured equity research reports in LaTeX format. Additionally, an interactive R Shiny application was developed with user registration, database integration, and automated report generation capabilities.

## Features

- Automated financial data extraction from APIs
- Data processing and analysis using R
- Automated generation of professional equity research reports
- LaTeX-based report formatting
- Interactive R Shiny application
- User registration and database integration

## Technologies

- R
- R Shiny
- LaTeX
- SQL Database
- Financial APIs

## Workflow

1. Financial data is retrieved from external APIs.
2. Data is cleaned and processed using R.
3. Analytical outputs are generated.
4. Reports are automatically compiled using LaTeX.
5. Users can access the reporting system through the R Shiny application.
